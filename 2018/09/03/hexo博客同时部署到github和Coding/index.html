hexo博客同时部署到github和Coding

2018-09-03
  • 博客开发

原文：http://www.voidcn.com/article/p-psfdomxz-yz.html

引言：
之前博客只部署到了github上，具体过程可参考基于hexo+github博客的搭建，我尝试同时部署到coding上。

1.打开Coding.net官网，注册个人账号。这里我直接关联的github账号。

2.新建项目
注意：项目名与注册用的账户名一致。

3.将根目录下的_config.yml的repo写成

4.部署即可

5.打开Coding.net的项目管理界面，打开代码->Pages服务，选择部署来源为master分支，然后保存即可。

6.访问地址：账户名.coding.me
